Basic JSONPath Examples:

Expression Result
$.store.book[*].title All book titles

üîç Advanced JSONPath Selectors & Filters

1. Filter by Condition

$.store.book[?(@.price > 15)]

Returns all books with price > 15

2. Filter by Multiple Conditions

$.store.book[?(@.category == 'fiction' && @.price < 15)]

Books in "fiction" category with price < 15

3. Access Last Element in Array

$.store.book[(@.length-1)]

Returns the last book

4. Access a Range of Elements

$.store.book[0:2]

First two books (index 0 and 1)

üí° Real-World JSONPath Examples
